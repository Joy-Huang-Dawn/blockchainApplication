# blockchainApplication
基于区块链的二手车交易系统
========================
链车APP是一个基于区块链的二手车市场交易平台（IOS平台），针对二手车销售者和消费者，聚合用户供求等信息，分类集中发布，方便用户交易的应用软件。通过利用区块链技术的特性能够最大程度上保证车况信息的真实性，能够为消费者提供真实有效的数据，解决二手车交易过程中的不公开、不透明、缺乏诚信度等问题缺陷。
